NITP(night in the pub) is game like reigns with cards. Writed in Unity. Translated into Russian only

To open you should add ".\NITP\programming\NITP_8" folder in unity like project
All assets(image, .ai files and sounds) in ".\NITP\assets" folder 

PS The project is abandoned due to lack of time and added to the github in the hope 
that at least someone can use it or even want to revive it

НИТП(ночь в пабе) это карточная игра похожая на reigns. Написанно на Unity. Переведено только на русский

Чтобы открыть Добавьте папку ".\NITP\programming\NITP_8" в unity как проект
Все асеты(картинки, файлы иллюсратора и звуки) в папке ".\NITP\assets"

Проект заброшен по причине отсутствия времени и добавлен в гитхаб в надежде на то, что
хоть кто то сможет его использовать или даже захочет оживить его
